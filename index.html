Scrolling Slider

A JavaScript plugin which moves list items left/right on hover or via the scrollbar.

Quick start

Simply load and run the script in any app

In the future I will make it fully CJS and AMD compatible

Features

  • Items can be any size and different sizes
  • Left/Right Controls are hidden when fully scrolled left/right
  • Depends on JQuery and JQuery Debounce Throttle
  • supports two layout modes: standard and mobile
  • Controls are hidden if the list contents are smaller then the container

This plugin was designed to solve the problem of using mouseenter as an event rather than click

It works inconjunction with scroll by managing the scroll position

To get started simply add the javascript file to yout project and be sure to load jQuery and debounce/throttle first.

Inlcude the styles from main.css and adjust the width and height of the list items as necessary

If using a module loader such as CJS or AMD you can load it in the same way you would any other plugin
